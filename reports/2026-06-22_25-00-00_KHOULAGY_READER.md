Implemented a full **الخولاجي المقدس** module: hero card on home, categorized hymn index (17 groups / 127 verses from Supabase), and a premium reader with auto-scroll, scroll rail, typography controls, dark spiritual mode, Arabic + Coptic display, and prev/next navigation — patterned after Agpeya/Bible reading UX.

### Reader Features

- `AutoScrollControls` — play/pause, speed, font, line spacing, dark mode
- `ChapterReadingScrollRail` — vertical scroll strip (Bible-style)
- `ReaderArticleProgress` — progress bar in header
- Section chips per verse/stanza
- Toggle Arabic / Arabic+Coptic (`ⲀⲂ` button)
- Swipe + prev/next between hymns
- Scroll position persisted per group (`localStorage`)
- Share / copy link

### Navigation Wiring

- Home journey card: **الخولاجي المقدس** → `/kholagy`
- Daily card «ترنيمة اليوم» → `/kholagy`
- Alpha Nav Hub item added
- Module gate: `agpeya` (same as liturgical prayers)

2. Coptic lines use monospace font; dedicated Coptic webfont not added (future polish).
3. No audio playback yet — text reader only.
